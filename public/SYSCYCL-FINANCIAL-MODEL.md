# PART 1: REVENUE PER BOTTLE ANALYSIS

## 1.1 PET Bottle Weight & Scrap Value (Ontario, 2026)

| Factor | Value | Source |
|--------|-------|--------|
| Average 500ml PET bottle weight | 10 grams (0.022 lbs) | Plastics Industry Data |
| Average 1L PET bottle weight | 22 grams (0.048 lbs) | Plastics Industry Data |
| Average mixed PET bottle weight | ~15 grams (0.033 lbs) | Our estimate |
| Ontario scrap PET price (June 2026) | $0.10 - $0.18 / lb | Ontario Scrap Metal Market |
| Conservative scrap price | $0.12 / lb | Conservative estimate |
| Optimistic scrap price | $0.18 / lb | Peak market rate |
| LCBO deposit (select bottles) | $0.10 / bottle | Ontario LCBO Program |

### Revenue Per Bottle Calculation

**Scenario A: Scrap Only (Non-LCBO bottles)**
```
Revenue per bottle = Weight × Scrap price per lb
                   = 0.033 lbs × $0.12/lb
                   = $0.004 per bottle (scrap only)
```

This is very low. **The key is VOLUME and LCBO deposits.**

**Scenario B: With LCBO Deposit Bottles**
```
If 40% of bottles are LCBO-eligible:
- 60% at scrap: 0.60 × $0.004 = $0.0024
- 40% at deposit: 0.40 × $0.10 = $0.0400
- Weighted average = $0.0424 per bottle
```

**Scenario C: Optimistic (High LCBO ratio)**
```
If 60% of bottles are LCBO-eligible:
- 40% at scrap: 0.40 × $0.004 = $0.0016
- 60% at deposit: 0.60 × $0.10 = $0.0600
- Weighted average = $0.0616 per bottle
```

### Realistic Revenue Per Bottle (Blended): ~$0.045 (4.5 cents)

### BREAK-EVEN ANALYSIS:
```
Break-even = when revenue covers all costs

Revenue per household per week: ~$0.45
Cost per volunteer per week: $50 (stipend + gas)

Break-even households = $50 ÷ $0.45 = 111 households per volunteer

With 1 volunteer: Need 111 households (currently at 90 — close!)
With 2 volunteers: Need 222 households
With 3 volunteers: Need 333 households
```
